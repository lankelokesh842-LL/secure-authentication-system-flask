# Secure Authentication System Using Flask and Bcrypt

## Overview
A secure web-based authentication system built using Flask, SQLite, SQLAlchemy, and Bcrypt.

## Features
- User Registration
- User Login
- Password Hashing with Bcrypt
- Session Management
- Logout Functionality
- SQL Injection Protection
- Password Validation

## Technologies Used
- Python
- Flask
- SQLite
- SQLAlchemy
- Bcrypt

## Installation

```bash
pip install flask flask-bcrypt flask-sqlalchemy
python app.py
```
